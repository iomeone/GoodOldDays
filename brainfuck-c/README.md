## About
Brainfuck-C is a [Brainfuck](http://en.wikipedia.org/wiki/Brainfuck) interpreter written in C.

## Usage
Compile using gcc and run:
```
./bf filename
```

## Examples
```
./bf hw.bf
```
```
curl -s "http://www.99-bottles-of-beer.net/download/1718" > bottles.bf
./bf bottles.bf
```
